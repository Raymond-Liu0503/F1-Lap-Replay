Setup Instructions

  1. Install Python backend:
    pip install -r requirements.txt
  2. Run the API server:
    python main.py
    or
    uvicorn main:app --reload
  3. Load drivers: Enter session details and driver code, then click "Load Driver"
  4. Compare multiple drivers: Load additional drivers by changing the driver code and clicking "Load Driver" again
  5. Click Play to watch the comparison with live telemetry!

üí° Tip: Compare up to 4 drivers simultaneously (e.g., VER, HAM, LEC, SAI) from the same session. Each driver gets a unique color and their own live telemetry display!

Note: First load may take 30-120 seconds. Data is cached locally and available from 2018 onwards.
